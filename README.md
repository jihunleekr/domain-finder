## 사용가능한 스크립트

### `npm run cli -- -t <TLD> -l <length>`

등록되지 않은 도메인을 찾습니다.

```
$ npm run cli -- -t .com -l 3 > .com.txt
```

## 제한사항

- 4글자 이상은 매우 느립니다.
- 일부 도메인의 경우 쿼리제한으로 검색도중 차단 당하는 경우도 있습니다.
